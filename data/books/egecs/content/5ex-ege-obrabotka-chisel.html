Краткие сведения

В этом задании нужно уметь с помощью Python писать короткие скрипты для перевода чисел из разных систем. Также важно внимательно читать алгоритм преобразования. В этом разделе на примере задачи увидим, как должен идти процесс решения. Внизу будет список необходимых команд на Python.

Разбираем на примере

На вход алгоритма подаётся натуральное число N. Алгоритм строит по нему новое число R следующим образом.

  1. Строится троичная запись числа N.
  2. Далее эта запись обрабатывается по следующему правилу:
    • если число N делится на 3, то к этой записи дописываются две последние троичные цифры;
    • если число N на 3 не делится, то остаток от деления умножается на 5, переводится в троичную запись и дописывается в конец числа.
  3. Полученная таким образом запись является троичной записью искомого числа R. Результат переводится в десятичную систему и выводится на экран.

Например, для исходного числа 1110 = 1023 результатом является число 1021013 = 30710, а для исходного числа 1210 = 1103 это число 110103 = 11110. Укажите минимальное число R, большее 150, которое может быть получено с помощью описанного алгоритма. В ответе запишите это число в десятичной системе счисления.

Заранее напишем ответ: 162

Готовое решение

# Решение задачи
def cc3(x): # Объявление функции для перевода в троичную систему
    s = "" # Создаём пустую строку для результата
    while x > 0: # Пока число больше нуля
        s = str(x % 3) + s # Добавляем остаток от деления в начало строки
        x = x // 3 # Делим число нацело на 3
    return s # Возвращаем результат - число в троичной системе

m = [] # Создаём пустой список для хранения подходящих чисел
for n in range(1, 300): # Перебираем числа от 1 до 299
    b = cc3(n) # Переводим текущее число в троичную систему
    if n % 3 == 0: # Если число делится на 3 без остатка
        b = b + b[-2:] # Добавляем две последние цифры к троичной записи
    else: # Иначе (если не делится на 3)
        b = b + cc3(n % 3 * 5) # Добавляем специальное троичное число
    r = int(b, 3) # Переводим полученную строку обратно в десятичное число
    if r > 150: # Если результат больше 150
        m.append(r) # Добавляем число в список

print(min(m)) # Находим и выводим минимальное число из списка

Что делает этот код!

Программа выполняет следующие шаги:

  1. Создает функцию для перевода чисел в троичную систему счисления
  2. Перебирает все числа от 1 до 299
  3. Для каждого числа выполняет специальное преобразование в троичной системе
  4. Переводит результат обратно в десятичную систему
  5. Отбирает только числа больше 150
  6. Находит и выводит наименьшее из отобранных чисел

Перевод из 10чной в любую другую

def convert_to_base(n, b):  # Объявляем функцию для перевода числа в другую систему счисления
  digits = "0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ"  # Строка со всеми возможными цифрами (до 36-ричной системы)
  result = ""  # Создаем пустую строку для результата

  while n > 0:  # Пока наше число больше нуля, выполняем цикл
    remainder = n % b  # Находим остаток от деления числа на основание системы
    result = digits[remainder] + result  # Берем символ из строки digits под номером остатка и добавляем его В НАЧАЛО результата
    n = n // b  # Делим число нацело на основание системы, уменьшая его
  print(result)  # Выводим результат на экран
  return result  # Возвращаем результат из функции (чтобы можно было использовать дальше в программе)

convert_to_base(112, 4)  # Вызываем функцию: переводим число 112 в четверичную систему (основание 4)
convert_to_base(2315, 9)  # Вызываем функцию: переводим число 2315 в девятиричную систему (основание 9)

Список команд, которые могут быть нужны

Команда Описание
bin() переводит число в двоичную систему счисления (основание 2). Результат включает префикс «0b»
oct() переводит число в восьмеричную систему счисления (основание 8). Результат включает префикс «0o»
hex() переводит число в шестнадцатеричную систему счисления (основание 16). Результат включает префикс «0x»
int(nmb,base) позволяет преобразовать строку, представляющую число в определённой системе счисления, в десятичное значение. Для этого нужно указать два аргумента:
nmb — строка с числом;
base — основание исходной системы счисления (от 2 до 36 включительно).
str() преобразует число в строку
float() преобразует число в дробное (с плавающей точкой)
round() округляет числа с плавающей точкой (поддерживает округление до заданного количества знаков после запятой)
number.count("1") считает количество 1 в строке числа. в кавычках ставит то, что считаем по количеству в строке (number хранит строку)
break команда располагается внутри цикла и останавливает его, когда до нее доходит очередь
[:-1] убирает последний символ в строке
[2:] убирает префикс при переводе (первые 2 символа)
"12" + "52" знак сложения при использовании со строками склеивает их
